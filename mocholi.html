Cronograma

1995

Apache:

Apache es un servidor web de código abierto que implementa el protocolo HTTP y fue lanzado inicialmente en 1995. Apache es usado para mantener páginas estáticas y dinámicas en Internet y está integrado en plataformas como LAMP, la BBDD de Oracle y WebSphere de IBM entre otros. Apache presenta entre otras características configurables, bases de datos de autenticación y negociado de contenido, pero fue criticado por la falta de una interfaz gráfica que ayude en su configuración. Esto no le impidió llegar al 70% de cuota de mercado total en 2005.

Guerra de los navegadores:

Durante 1995, la Internet empezó a ganar popularidad y entre los navegadores Netscape era la opción número 1, en comparación a Mosaic y a Lynx. En un principio, Microsoft no vio el potencial en la web, pero tras ver el posible potencial comercial, compró licencias de Mosaic y sacaron las dos primeras versiones de Internet Explorer. Esto provocó que Netscape y Microsoft empezasen a priorizar nuevas funcionalidades en lugar de los estándares y el arreglo de fallos.

Internet Explorer:

Creado en 1994 y basado en Mosaic, su primera aparición fue como complemento para sistemas Windows 95. Meses después se lanzó la versión 1.5 que integraba el navegador de forma gratuita, lo que causó una demanda por parte de Spyglass Inc. Pagando una indemnización de millones de dólares, sacaron la versión 2.0 con administración de frames, SSL, cookies y el protocolo NNTP.

Opera:

Opera nació como un proyecto de Telenor en 1995 y posteriormente fue propiedad de Opera Software el mismo año. Su primera versión no salió al público, pero tenía una interfaz de múltiples documentos y una barra lateral para facilitar la navegación.

Java:

Java fundado por Sun Microsystems, es un lenguaje orientado a objetos usado para crear servidores de aplicaciones web. Posee la JVM, que interpreta código binario de las instrucciones y las compila a java.

PHP:

PHP es un lenguaje adaptable facilmente al desarrollo web. Hasta 2014 no tenía estándares más allá de su fundación.

Python:

Creado en 1991, es un lenguaje interpretado con enfasis en la legibilidad de su código. Es multiparadigma porque soporta parcialmente diversos paradigmas como el de objetos, imperativa y funcional.

MySQL:

Es un sistema de gestión de BBDD en lenguaje SQL. Es uno de los componentes de las pilas WAMP y LAMP.

1996

Internet Explorer:

Lanzado en 1996 la versión 3.0, seguía evadiendo pagos adicionales a Spyglass. Esta versión era más parecida a Netscape y contenía soporte de plugins, ActiveX, frames, una versión de JS llamada JScript y fue el primero en introducir CSS a los navegadores.

1997

EcmaScript:

Lanzada la versión 1.0. ECMAScript es una especificación del lenguaje de programación JS, siendo este último creado por Netscape. A diferencia de JS, EcmaScript está más inspirado en Java y C.

Internet Explorer:

Lanzado en 1997 la versión 4.0, introdujo el predecesor al RSS, Active Channel, además de HTML dinámico y Outlook. A pesar de estar integrada en Windows 98, funcionaba sin problemas en 95 y NT de forma gratuita.

Netscape:

Lanzado en 1997 la versión 4.0, fue un competidor más rápido que Explorer, pero eso no convenció a empresas como Apple la adopción de IE en sus Mac OS.

Guerra de los navegadores:

Durante 1997, diversas páginas promocionaban versiones concretas de ambos navegadores, alegando que se veían mejor en ellos. La ventaja inicial de Netscape fue abolida por los recursos de Microsoft y sus tácticas como atacar las ganancias de Netscape, firmar contratos con fabricantes y mayor flexibilidad para modificar su navegador. Además IE venia incluido en cada ordenador y con los beneficios integró las funcionalidades rápidamente.

Netscape Application Server:

Netscape Application Server era un IDE para desarrollar y ejecutar aplicaciones orientadas a transacciones en la web.

Java:

Versión 1.1 lanzada. Incluye POJOs, JDBC e invocación remota de métodos.

1998

Netscape:

Tras el fracaso de la versión 4 de su navegador, es comprada por America Online y publican su código como abierto. Los esfuerzos futuros se centran en crear Firefox a partir de ese código.

Java:

Versión 1.2 lanzada. Incluye principalmente JavaSwing y la reflexión en programación.

EcmaScript:

Lanzada la versión 2.0. Se adaptó al estándar ISO/IEC 16262.

1999

EcmaScript:

Lanzada la versión 3.0. Se agregaron expresiones regulares, mejor manejo de strings, nuevo control de declaraciones, manejo de excepciones con try/catch, definición más estricta de errores, formato para la salida numérica

Tomcat:

Es un contenedor de servlets creado por Apache.

Apache:

Apache es un contenedor es una organización sin fines de lucro para dar soporte a los proyectos creados por Apache. Basados en la licencia de software libre, cuenta patrocinadores conocidos como Google, Microsoft, Facebook entre otros.

phpMyAdmin:

phpMyAdmin es una herramienta en PHP para manejar la administración de BBDD a través de una página web. Es uno de los pilares de las pilas LAMP y WAMP.

2000

Java:

Versión 1.3 lanzada. Incluye un depurador, una versión actualizada de la máquina virtual y permite introducir sonido.

Ant:

Es una herramienta usada para tareas repetitivas. Requiere Java y tiene la ventaja de no depender del shell, sino que usa archivos XML.

2001

Internet Explorer:

Lanzada la versión 6.0. Incluido en Windows XP, brindaba mayor privacidad y seguridad, pero sufrió de graves problemas durante su duración. A partir de aquí, se sacarían versiones correspondientes a cads nuevo sistema operativo.

Hibernate:

Es una herramienta de mapeo objeto-relacional que facilita el mapeo de atributos entre una base de datos y el modelo de objetos de Java con XML o con beans.

Eclipse:

Es una plataforma de software para crear aplicaciones de código enriquecido. Frecuentemente es usado como un IDE

IntelliJ:

Es un IDE. IntelliJ tiene más recursos y soporta de forma nativa más lenguajes que Eclipse. También ofrece mejor depuración.

2002

Java:

Versión 1.4 lanzada. Incluye regex, encadenar excepciones, API de logging, parseador XML y mayor seguridad.

Maven:

Es una herramienta para gestionar y construir proyectos en java con dependencias en el pom.xml. Está construido sobre una arquitectura de plugins que permite utilizar cualquier aplicación.

Struts:

Es una herramienta para desarrollar aplicaciones web en MVC en Java.

Spring:

Spring es un framework para desarrollar aplicaciones y contenedor de inversión de control principalmente en Java

2003

Safari:

La primera versión de Safari fue lanzada en 2003 hecha para dispositivos Mac, sustituyendo a Internet Explorer como navegador predeterminado.

2004

Mozilla Firefox:

Creado por antiguos empleados de Netscape, fue lanzado Firefox. Una de sus principales características era la inclusión de componentes, extensiones realizadas por terceros. Fue el primer competidor de IE en el siglo XXI.

Java:

Versión 5 lanzada. Incluye código genérico con conversión de tipos, metadatos, enumeraciones, mejores bucles y número de argumentos variable.

Spring Security:

Un framework derivado de Spring que ofrece autenticación, autorización y medidas de seguridad.

2005

Symfony:

Es un framework de PHP según el modelo MVC. Compatible con la mayoría de gestores de datos.

Cake:

Es un framework de PHP basado en Ruby on Rails.

Django:

Framework en Python que respeta el MVC. Su meta es crear sitios complejos a través de componentes.

2006

Jquery:

Es una librería creada por John Resig que simplifica el HTML, permite manipular el DOM, maneja eventos y permite AJAX. También se pueden crear abstracciones fácilmente para crear páginas dinámicas.

Java:

Versión 6 lanzada. Incluye combinación con lenguajes como PHP o Ruby, mejoras de rendimiento y gráficas y mayor soporte para la web.

JPA:

Java Persistence API es un framework que maneja datos relacionales usando Java. Conserva la orientación a objetos interactuando con una BD y el uso de POJOS.

Struts:

Versión 2.0 que simplifica tareas comunes y mayor integración en AJAX. Ha sido criticado por fallos que permiten la ejecución de código arbitrario.

CodeIgniter:

Framework de PHP usado para construir páginas web dinámicas. Es más rápido que la mayoría de entornos.

Liquibase:

Liquibase es una biblioteca independiente de la base de datos y de código abierto para el seguimiento, la gestión y la aplicación de cambios en el esquema de la base de datos. Fue creada en 2006 para facilitar el seguimiento de los cambios en las bases de datos, especialmente en un entorno ágil de desarrollo de software.

Zend:

Framework de PHP orientado totalmente a objetos. Sus componentes son independientes de otros.

2007

Gradle:

Es un sistema de automatización de construcción de código de software construido sobre Ant y Maven e introduce un lenguaje llamado DSL en vez de XML. Creado para proyectos conjuntos de gran envergadura.

Git:

Es un software de control de versiones creado por Linus Torvalds. Lleva el registro de cambios de archivos y coordinar el trabajo en archivos compartidos en un repositorio de código.

Sonarqube:

Es una herramienta para evaluar código fuente y da consejos de cómo mejorarlo.

2008

V8:

Motor creado por Google y lanzado junto a la primera versión de Chrome. Se puede usar tanto para cliente como para servidor.

Google Chrome:

Navegador creado por Google. El código de Chrome proviene de una versión modificada de Chromium, su variante libre. Principalmente es estable, seguro y veloz. Progresivamente obtuvo la mayor cuota de mercado, incluyendo que sus competidores estén basados en Chromium.

GSON:

Es una biblioteca de código abierta creada por Google para java que serializa y deserializa objetos y los representa en JSON.

Yii:

Framework de PHP basado en componentes y de alto rendimiento.

Github:

Es una plataforma de desarrollo colaborativo para alojar proyectos con el sistema de controlador de versiones Git.

2009

EcmaScript:

Versión 5.0, siendo más estricto, adquiere getters y setters, soporte para JSON y modificaciones a los objetos.

Underscore:

Es una librería de JS creada por Jeremy Ashkenas y fue diseñada para la programación funcional.

NodeJS:

Es un entorno servidor-cliente multiplataforma basado en JS. Creado inicialmente para servidores web.

2010

Backbone:

Es un framework de JS creado por Jeremy Ashkenas basado en el MVC. Diseñado para aplicaciones de una sola página.

AngularJS:

Es un framework de JS creado por Google basado en el MVC. Diseñado para aplicaciones de una sola página. También se puede combinar con nodeJS, Express y la BBDD MongoDB. Su comportamiento se basa en las directivas ng que permiten manipular el DOM.

Express.js:

Es un framework de backend para JS diseñado para APIs y aplicaciones web. Es el framework por excelencia para servidores en Node.

2011

React:

Es un framework de JS mantenido por Facebook basado en el MVC o MVVM. Diseñado para aplicaciones con datos que siempre cambian. Diseñado para interfaz de usuario, fue usado para el feed de Facebook y en 2013 fue liberado al público.

Java:

Versión 7 lanzada. Incluye soporte de XML, soporte para closures, superpaquetes y anotaciones.

Laravel:

Framework para trabajar con PHP de forma simple, evitando código espagueti.

Travis CI:

Es un servicio de integración continua usado para testear software alojado en Github y Bitbucket. Fue el primer servicio de integración continua gratuito.

2012

Typescript:

Es un lenguaje extendido de Javascript que añade tipos estáticos y clases. Pensado para grandes proyectos.

Hikari:

Es una librería Java para crear pool de conexiones muy ligera. Construida fácilmente con una dependencia de Maven.

2013

Vue.js:

Creado como alternativa liviana y flexible a AngularJS basado en el modelo MVVM. Vue se centra en usar HTML en vez de Javascript con modelos y componentes.

Flux:

Predecesor de Redux. Es una arquitectura para manejar el flujo unidireccional de datos en frontend. Creado por Facebook.

JHipster:

Es un generador de aplicaciones y microservicios para desarrollar rápidamente aplicaciones en Angular, React o Spring.

Docker:

Automatiza el despliegue de aplicaciones en contenedores de software, proprocionando abstracción y automatización. Evita la sobrecarga del sistema al no depender de máquinas virtuales.

2014

Java:

Versión 8 lanzada. Incluye mejoras generales, permite expresiones lambda para programación funcional y nueva API para manejar fechas.

Spring Boot:

Permite crear proyectos basados en Spring fáciles de ejecutar. Suele llevar su propio Tomcat.

npm:

Sistema gestor de paquetes para NodeJS.

2015

RxJS:

Son una serie de librerías para componer eventos asíncronos a través de observables reactivos para JS, que tiene el tipo Observable.

EcmaScript:

La versión 6.0 agrega clases, módulos, iteradores de bucles, generadores de expresión, funciones de dirección, datos binarios, colecciones, y metaprogramación haciéndolo más parecido a lenguajes orientados a objetos.

Redux:

Redux es una librería para manejar el estado de las aplicaciones. Posee una API simple y fue inspirada por flux.

Spring Roo:

Es una herramienta para desarrollar rápido proyectos en Java. Combina Spring, JPA, Thymeleaf, Maven y AspectJ.

Visual Studio Code:

Es un editor creado por Microsoft para Windows, Linux y Mac. Incluye depuración, control de Git, resaltado, refactorización y es altamente personalizable.

2016

Angular:

Es un framework de JS creado por Google basado en directivas que alteran el DOM y sucesor de AngularJS. Diseñado para aplicaciones de una sola página. a diferencia de AngularJS, posee un mayor rendimiento y se centra en componentes.

2019

GitHub Actions:

Automatiza y ejecuta flujos de trabajo directamente en Github.
